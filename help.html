IMPORTANT NOTE: If a game isn't registering inputs, check the hold interval, it may be too quick that the game doesn't even register it.
(Optional): When something is optional, it defaults to 0, so if you need to fill in an argument, put in 0, otherwise you can leave it blank.
Coordinates: If coordinates are set to 0,0 or blank, they will default to mouse cursor.
Interval: Most intervals default to 10ms if less than 10ms. (Computers only register something around 15-20ms)

Command: MOUSE button interval clickCount X Y
Example: MOUSE left 100 1 1400 475
Explanation: Plays a mouse button click.

Details:

  • button: The mouse button (left, right, middle)
  • interval: Duration the button is held down (Optional)
  • clickCount: Amount of clicks performed per action (Optional)
  • X, Y: Coordinates (Optional)
Command: MOUSE button state X Y
Example: MOUSE left down 1400 475
Explanation: Holds/unholds mouse button.

Details:
  • button: The mouse button (left, right, middle)
  • state: The state of the button (up, down)
  • X, Y: Coordinates (Optional)

Command: MOVE X Y
Example: MOVE 1400 475
Explanation: Teleport cursor to location.

Details:

  • X, Y: Coordinates

Command: MOVETO X Y
Example: MOVETO 1400 475
Explanation: Simulates moving cursor to location.

Details:

  • X, Y: Coordinates

Command: VSCROLL num
Command: HSCROLL num
Example: VSCROLL -1
Example: VSCROLL 1
Explanation: Scrolls mouse wheel based on specified number

Details:

  • num: The amount of scrolls, 1 or -1 will scroll in each direction one notch

Command: RANDOMMOUSE button lowMs highMs clickCount X Y
Example: RANDOMMOUSE left 100 300 1 1400 475
Explanation: Plays a mouse button click with a random hold interval between lowMs and highMs

Details:

  • button: The mouse button (left, right, middle)
  • interval: Duration the button is held down (Optional)
  • clickCount: Amount of clicks performed per action (Optional)
  • X, Y: Coordinates (Optional)

Command: KEYBOARD key interval
Example: KEYBOARD E 60
Explanation: Plays a keyboard key.

Details:

  • key: The keyboard key (LeftShift, e, F5) (Use recording feature to figure out keys)
  • interval: Duration the button is held down (Optional)

Command: WRITE string
Example: WRITE this is a message
Explanation: Will send a string to the cursor/field that is focused.

Details:

  • string: Any kind of text

Command: HOLD key
Command: UNHOLD key
Example: HOLD e
Explanation: Holds/Unholds a keyboard key. Useful for gaming, but not for text input.

Details:

  • key: The keyboard key

Command: RANDOMKEYBOARD key lowMs highMs
Example: RANDOMKEYBOARD e 100 200
Explanation:
Plays a keyboard key with a random hold interval between lowMs and highMs
Details:

  • key: The keyboard key
  • lowMs: Lowest number random will pick
  • highMs: Highest number random will pick

Command: WAIT interval
Example: WAIT 100
Explanation: Pauses the script for the specified time in milliseconds.

Details:

  • interval: Duration of wait

Command: RANDOMWAIT lowMs highMs
Example: RANDOMWAIT 100 250
Explanation:
Pauses the script for the specified time randomly between lowMs and highMs in milliseconds.
Details:

  • lowMs: Lowest number random will pick
  • highMs: Highest number random will pick

Command: LOOP amt
Command: END
Example: LOOP 15
Explanation: A basic loop that will loop through the specified amount.

Details:

  • amt: Loop iterations

Full Example:
LOOP 15
MOUSE left down
WAIT 100
MOUSE left up
END

Command: IF condition value
Command: END
Example: IF window firefox
Explanation: Checks condition and if true, runs commands inside.

Details:

  • condition: Condition to check
  • value: Value to check

Full Example:
IF color #FFFFFF
MOUSE left down
WAIT 100
MOUSE left up
END

Command: OPENURL url
Example: OPENURL https://www.google.com/
Explanation: Opens a new window with the specified URL through your default browser.

Details:

  • url: A website URL

Command: RUN process
Example: RUN C:\Path\To\File\process.exe
Explanation: Runs a process. Such as an AHK file or an exe file.

Details:

  • process: The full file path to an executable file
